# 4주차) 크리스마스 프로모션

---
## 프로젝트 개요
### 1. 프로젝트 목표
- 크리스마스 대비 이벤트 진행을 통한 최고 수익률 돌파 및 고객수 확보
### 2. 프로젝트 기간
- 2023년 11월 10일(금) ~ 2023년 11월 15일(수)
### 3. 프로젝트 범위
- 혜택 내역 및 할인 가격 안내 기능
- 할인 전 총주문 금액에 따른 증정 메뉴 안내 기능
- 총혜택 금액에 따른 이벤트 배지 안내 기능
### 4. 기술 스택
- 프로그래밍 언어 : `Java`
- IDE(통합 개발 환경) : `IntelliJ`
- API : Java 내장 API, 우아한테크코스 제공 커스텀 API
